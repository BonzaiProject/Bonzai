Requisiti e Dipendenze

Requisiti di sistema:

  • PHP 5.x

L'unica dipendenza è il compilatore PHP bcompiler utilizzato per generare il bytecode.

Installazione

Nessun processo di installazione è necessario. Si può semplicemente scaricare il codice sorgente ed eseguirlo così com'è.

è inoltre possibile utilizzare PEAR per installare il pacchetto, per installare usando il nostro repository PEAR seguire questi due passi:

  • pear channel-discover pear.bonzai-project.org
  • pear install bonzai/Bonzai

Configurazione

Non è necessaria alcuna configurazione.

                Utilizzo:
                ./bin/bonzai-cli [OPZIONI]... [FILES|CARTELLE]...

                Opzioni:
                    -b, --backup         Backup del file originale, generare un file con
                                         estensione .bak
                    -d, --dry            Esegue un giro di prova senza effettuare modifiche
                    -r, --report         Genera un rapporto completo
                        --colors         Usare i colori nei messaggi in uscita
                        --log=   Salva i messaggi di esecuzione in formato testuale
                        --stderr         Scrive nello STDERR invece dello STDOUT
                    -q, --quiet          Modalità silenziosa. Non scrive nulla
                    -h, --help           Mostra l'aiuto
                    -v, --version        Mostra la versione

                Segnalare i problemi a info@bonzai-project.org
