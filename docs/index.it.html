Requisiti e Dipendenze

Requisiti di sistema:

  • PHP 5.x

L'unica dipendenza è il compilatore PHP bcompiler utilizzato per generare il bytecode.

Installazione

Nessun processo di installazione è necessario. Si può semplicemente scaricare il codice sorgente ed eseguirlo così com'è.

Se volete installarlo come applicazione di sistema lanciate il comando sudo make install.

Disinstallazione

Per disinstallare l'applicazione lanciate il comando sudo make uninstall.

Configurazione

Non è necessaria alcuna configurazione.

Traduzioni

Per tradurre la documentazione bisogna creare un nuovo file partendo da docs/index.en.html e traducendolo nel file docs/index.[CODE].html dove [CODE] è il codice di due lettere della lingua.

Per tradurre lo script bisogna copiare il file messages.pot nella file locales/[CODE]/LC_MESSAGES/messages.po (ovviamente la cartella deve essere creata prima) dove [CODE] è il codice estesto della lingua (es: en_US). Successivamente si può utilizzare un tool per modificare il file (es: poEdit) oppure si può modificare manualmente. Andrà poi generato il file messages.mo.
